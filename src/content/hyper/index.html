Hyper

Install using hpm

Install via hpm command line:

$ hpm install hyper-dracula

Install using config file

Add hyper-dracula to the plugins list in your ~/.hyper.js config file.

plugins: [
	"hyper-dracula"
]

Activating theme

  1. Start Hyper
  2. Go to View -> Full Reload
