Facebook Messenger

Install manually

  1. Go to the [google chrome web store](https://chrome.google.com/webstore/detail/dracula-theme-for-faceboo/ijphgphffpboghbejdbjjoddacphjafh)
  2. Click on Add to Chrome
  3. Click on Add extension
  4. Navigate to https://www.messenger.com and click on the dracula icon to toggle the theme

Note: The theme is also available on the Desktop Themes Channel
