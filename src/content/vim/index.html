Vim

Install using Git

If you use vim + pathogen:

cd ~/.vim
git submodule add git@github.com:dracula/vim.git bundle/dracula

If you use vim + vundle:

Plugin 'dracula/vim', { 'name': 'dracula' }
:PluginInstall

If you use vim-plug (`as` will install the plugin in a directory called 'dracula' instead of just 'vim'):

Plug 'dracula/vim', { 'as': 'dracula' }
:PlugInstall

If you aren't so clever just move the dracula.vim file into ~/.vim/colors and add the following lines into your vimrc file:

syntax on
colorscheme dracula
