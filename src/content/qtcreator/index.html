Qt Creator

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/dracula/qtcreator.git ~/.config/QtProject/qtcreator/styles

Install manually

  1. Download using the GitHub .zip download option and unzip them
  2. Move the dracula.xml file to ~/.config/QtProject/qtcreator/styles

Activating theme

Go to Qt Creator -> Preferences..., click in the Text Editor tab, and select Dracula in the Color Scheme.
