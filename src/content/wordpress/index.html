WordPress

Install manually

  1. Add the folder Dracula-custom-admin-color-schemes inside your WordPress plugins folder ./wp-content/plugins/
  2. Login to the admin dashboard
  3. Navigate to the plugins tab
  4. Activate Dracula theme plugin
  5. All done.
